# CGPA Calculator

# Project Description

This is a C++ program that calculates a student's CGPA based on grade points and credit hours of multiple courses.

 Formula Used

CGPA = Sum(Grade Point × Credit Hours) / Sum(Credit Hours)

# Example

| Course | Grade Point | Credit Hours |
| ------ | ----------- | ------------ |
| C++    | 9           | 4            |
| DSA    | 8           | 3            |
| DBMS   | 10          | 4            |
| OS     | 7           | 3            |

Total Credits = 14

Total Grade Points = 121

CGPA = 121 / 14 = 8.64

# Algorithm

1. Start the program.
2. Input the number of courses.
3. Initialize total credits and total grade points to 0.
4. For each course:

   * Enter course name.
   * Enter grade point.
   * Enter credit hours.
   * Calculate grade point × credit hours.
   * Add values to totals.
5. Calculate CGPA.
6. Display course details, total credits, total grade points, and CGPA.
7. End the program.

# Features

* Accepts multiple course inputs.
* Calculates total credits automatically.
* Computes total grade points.
* Displays final CGPA accurately.
* User-friendly interface.
